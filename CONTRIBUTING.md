# Contributing

Thank you for improving Actaro. Open an issue before large changes, keep the SDK focused, and include tests for behavior changes.

1. Use Node.js 20 or newer and run `npm install`.
2. Create a focused branch and write clear, strict TypeScript.
3. Run `npm run lint`, `npm test`, and `npm run build`.
4. Submit a pull request describing motivation, behavior, and tests.

By participating, you agree to follow the Code of Conduct.
